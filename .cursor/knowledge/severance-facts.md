# Severance / Lumon — Facts & Flavor Reference

Canonical flavor for this repo. Prefer show-accurate terms; invent only when mapping outie habits → innie protocols. Do not claim unused plot beats as app features.

## Lumon Industries
- Mega-corp with a cult of personality around founder **Kier Eagan**
- Public face is wellness / biotech; severed floor is opaque to innies
- Motto energy: devotion, compliance, “the work is mysterious and important”
- Handbook / compliance language is dry, ceremonial, slightly threatening

## Severance procedure
- **Innie** — workplace consciousness; no memory of the outside
- **Outie** — outside life; no memory of work
- Elevator / code detectors mark the boundary
- Innies experience only the severed floor unless granted perks (waffle, MDE, wellness)

## Macrodata Refinement (MDR)
- Office on a severed floor (this app: Floor 7 / MDR)
- Refiners sort “scary” floating numbers into **Four Tempers** bins:
  - **Woe (WO)**
  - **Frolic (FC)**
  - **Dread (DR)**
  - **Malice (MA)**
- Progress fills a **file**; completion is celebrated (Waffle Party)
- Terminal aesthetic: CRT/phosphor green, clinical UI, quadrants

## Key locations / departments (name-drop carefully)
- **MDR** — Macrodata Refinement (primary setting)
- **O&D** — Optics & Design
- **Break Room** — punitive / “wellness” interrogation space (tone: dread, not comedy spam)
- **Wellness** — sessions with Ms. Casey energy (calm, uncanny)
- **Perpetuity Wing / Kier shrine** — mythology, not a daily UI panel

## Recurring objects & perks
- **Finger trap** — corporate “fun” / stress object
- **Eraser (Helly)** — contraband / idle desk object
- **Music Dance Experience (MDE)** — rare high-reward spectacle
- **Waffle Party** — file-complete celebration; prestige cycle in this app
- **Melon bar / egg bar / coffee cozy** — perk / incentive flavor
- **Ballast / ceiling grid** — MDR lighting; faults feel departmental

## Tempers (emotional engine)
- Numbers are sorted by emotional “feeling,” not arithmetic explanation
- UI should make dominant temper visible (quadrant glow, digit behavior)
- Healthy metrics → Frolic; missed protocols / low sustenance → Woe / Dread; breach → Malice
